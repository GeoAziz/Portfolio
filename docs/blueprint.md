# **App Name**: Personal OS

## Core Features:

- Identity Statement: Display a concise and informative personal identity statement.
- Skill Orbit Visualization: Showcase skills in an interactive orbit, sourced from skills.json.
- Project Inspector: Display interactive project details, sourced from JSON files.
- Page-Specific Showcases: Dedicate pages to showcase software systems, AI/ML projects, hardware projects, research, open-source contributions, and resume.
- Command Palette Navigation: Implement a command palette (Ctrl+K or /) for quick navigation.
- Model Viewer: Display 3D models, or static images as a fallback.

## Style Guidelines:

- Background color: Dark gray (#0A0A0A) to create an engineered, calm atmosphere.
- Surface colors: Darker gray (#111111) and even darker gray (#1A1A1A) to distinguish page sections.
- Text colors: Light gray (#EAEAEA) for primary text and muted gray (#9A9A9A) for secondary text.
- Accent colors: Aqua (#65D6FF) and Yellow (#FFD23C) for future-aqua energy and important signals.
- Heading font: 'Space Grotesk' (sans-serif) for a bold, modern feel.
- Body font: 'Inter' (sans-serif) for a sharp, readable presentation.
- MotionFade: Use opacity and translateY for smooth fade-in effects with easeInOut easing.
- Active link underline in navigation expands smoothly on hover.
- Page transitions use fade-out/fade-in effects.
- Base spacing follows a 4px grid for a precise and engineered feel.
